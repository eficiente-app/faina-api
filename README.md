# faina-api
Faina - API

## Instalação

``` bash
$ git clone git@github.com:eficiente-app/faina-api.git
$ cd faina-api
$ cp .env.example .env
$ yarn
$ yarn build
$ yarn lint
$ yarn dev
```

``` bash
Output:
  [1] [nodemon] starting `node dist/index.js`
  [1] Servidor inicializado na porta 3000!
```
